Advanced Game Settings


Dam Ka! allows you to play Checkers, on non-standard boards.
In order to change the board characteristics, select the Advanced option from the New Game dialog.
You will receive the following screen:



Using this screen you can select the board dimensions.
The following boards are supported:
  • Mini Checkers: 6x6 Board.
  • Traditional Checkers: 8x8 Board.
  • Large Checkers: 10x10 Board.

You may also select the number of rows peons to begin with, by selecting the number of starting rows. The following options are available:
  • One row (4 Peons on the 8x8 board)
  • Two rows (8 Peons on the 8x8 board)
  • Three rows (12 Peons on the 8x8 board, This is the default option)
  • Four rows (This option is only available for the Large Checkers game.)
